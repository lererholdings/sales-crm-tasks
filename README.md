# sales-crm-tasks

A personal sales task management system for tracking pre-sale and post-sale activities across customers, prospects, partners, and distributors.

## Stack

| Layer | Technology |
|---|---|
| Frontend | React (Vite) |
| Hosting | Vercel |
| Database | Supabase (Postgres) |
| Auth | Clerk |
| Version control | GitHub |

## Repository structure

```
sales-crm-tasks/
  docs/
    design.md        ← full PRD + Technical Design Document
  db/
    schema.sql       ← Postgres DDL (run once in Supabase SQL editor)
  frontend/          ← React app (to be scaffolded)
  api/               ← Vercel serverless functions (to be scaffolded)
```
